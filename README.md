# Wheather Location

A simple application using Reactjs and Weather API

**To Run**

<p>~ cd wheater-location</p>

Important!!!
Needs a ".env" file on the root directory:
REACT_APP_OPEN_WHEATHER_KEY=< your wheather key >

<p>~ yarn install</p>
<p>~ yarn start</p>
